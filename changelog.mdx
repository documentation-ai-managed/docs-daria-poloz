## Upgrade Guide

Follow these steps to upgrade Daria Poloz smoothly.

<Steps>
  <Step title="Backup Data" icon="database">
    Export your projects:
    
````bash
daria export --all-projects > backup.json
````
  </Step>
  <Step title="Update Package" icon="download">
    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm install daria-poloz@latest
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn add daria-poloz@latest
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Migrate Config" icon="settings">
    Convert old YAML configs:
    
````bash
daria migrate-config old-config.yaml new-config.json
````
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart and verify:
    
````bash
daria restart && daria status
````
  </Step>
</Steps>

<Callout kind="tip">
  Test upgrades in a staging environment first.
</Callout>
