## Overview

Daria Poloz provides a centralized documentation space where you organize, manage, and collaborate on all your project documentation. You create structured pages, embed code examples, and share knowledge efficiently without switching tools. This platform supports MDX for rich, interactive docs with components like tabs and steps.

<Callout kind="info">
Daria Poloz uses the brand color `#3B82F6` for primary elements. Apply it in your custom themes or UI customizations to maintain consistency.
</Callout>

## Key Benefits

Daria Poloz streamlines your documentation workflow. You benefit from version control, real-time collaboration, and seamless integration with your existing stack.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com` with your email.
  </Step>
  <Step title="Set Up Space" icon="settings">
    Create a new documentation space named "My Project Docs".
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Click "New Page" and use the MDX editor to draft your introduction.
  </Step>
  <Step title="Publish" icon="globe">
    Preview and publish. Share the link with your team.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
```javascript
// Embed a Daria Poloz page in your app
const embedDoc = (pageId) => {
  const iframe = document.createElement('iframe');
  iframe.src = `https://docs.example.com/embed/${pageId}`;
  iframe.style.border = 'none';
  document.body.appendChild(iframe);
};
```
```python
# Python example to fetch page content via API
import requests

response = requests.get(
  'https://api.example.com/docs/pages/my-intro',
  headers={'Authorization': 'Bearer YOUR_API_KEY'}
)
print(response.json())
```
</CodeGroup>
