## Prerequisites

Before you begin, ensure you have:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: Existing documents in Markdown, PDF, or Google Docs format for import

</Callout>

## Create Your Account and Log In

Follow these steps to sign up and access Daria Poloz.

<Steps>
  <Step title="Visit the Dashboard" icon="globe">
    Open your browser and navigate to `https://app.dariapoloz.com`.
  </Step>
  <Step title="Sign Up" icon="user-plus">
    Click **Sign Up** and enter your email, name, and a strong password. Verify your email via the confirmation link sent to your inbox.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard, enter your credentials, and click **Log In**. You'll land on the main workspace.
  </Step>
</Steps>

## Set Up Your First Documentation Space

Organize your projects by creating a dedicated space.

<Steps>
  <Step title="Create New Space" icon="plus">
    From the dashboard, click **New Space** in the top-right corner.
  </Step>
  <Step title="Configure Space" icon="settings">
    Enter a name like `My Project Docs`, select a color (e.g., #3B82F6), and choose visibility: public, private, or team-shared.
  </Step>
  <Step title="Save and Enter" icon="check-circle">
    Click **Create**. Your space opens with a blank page ready for content.
  </Step>
</Steps>

## Import Existing Documents

Bring in your current docs seamlessly. Choose your preferred method.

<Tabs>
  <Tab title="Markdown Files" icon="file-text">
    Upload `.md` or `.mdx` files directly.

    <Steps>
      <Step title="Upload">
        In your space, click **Import** > **Markdown**. Select files from your device.
      </Step>
      <Step title="Review">
        Preview and edit imported pages before publishing.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Google Docs" icon="file-text">
    Connect your Google account for one-click import.

    ```bash
    # Example API integration for advanced users
    curl -X POST https://api.dariapoloz.com/v1/import/google \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{"docId": "1abc123def456ghi789", "spaceId": "space-uuid"}'
    ```
  </Tab>
  <Tab title="PDFs" icon="file">
    Convert PDFs to editable Markdown.

    <Callout kind="tip">
      PDFs convert automatically with 95% accuracy for text-heavy docs.
    </Callout>
  </Tab>
</Tabs>

<CodeGroup tabs="cURL,JavaScript">
```bash
curl -X POST https://api.dariapoloz.com/v1/spaces \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Imported Docs",
    "color": "#3B82F6"
  }'
```

```javascript
const response = await fetch('https://api.dariapoloz.com/v1/spaces', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My Imported Docs',
    color: '#3B82F6'
  })
});
```
</CodeGroup>
